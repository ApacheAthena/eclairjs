fill(value, cols) → {DataFrame}

Returns a new DataFrame that replaces null or NaN values.
Parameters:
Name Type Description
value number | string | object If number replaces null or NaN values in numeric columns with `value`. If string replaces null values in string columns with `value`. If object, the object is expected to be a HashMap, the key of the map is the column name, and the value of the map is the replacement value. The value must be of the following type: `number`or `String`.
cols string[] Optional, replaces null or NaN values in specified columns. Not valid when value is a map. If a specified column type does not match the values type, it is ignored.

replace(col, replacement) → {DataFrame}

Replaces values matching keys in `replacement` map with the corresponding values. Key and value of `replacement` map must have the same type, and can only be numbers or strings. If `col` is "*", then the replacement is applied on all string columns or numeric columns.
Parameters:
Name Type Description
col string name of the column to apply the value replacement
replacement object value replacement map, as explained above

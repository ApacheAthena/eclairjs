StructField

A field inside a StructType.

new StructField(name, dataType, nullable, metadata)

Parameters:
Name Type Description
name string The name of this field.
dataType DataType The data type of this field.
nullable boolean Indicates if values of this field can be null values.
metadata Metadata The metadata of this field. The metadata should be preserved during transformation if the content of the column is not modified, e.g, in selection.
